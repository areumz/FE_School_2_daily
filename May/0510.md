* 재귀함수   
실무에서 많이 사용하지는 않음   
반복문으로 구현할 수 있는 것은 재귀함수로 모두 구현 가능   
재귀함수로 구현가능한 것은 반복문으로 대부분 구현 (복잡도를 증가시키면 모두) 가능   
재귀함수는 꼭 종료 조건 있어야함   

```
function factorial(n) {
    
    if (n <= 1) {

        return n
    
            }
    
    return n * factorial(n-1)

}
```

위의 재귀함수
이렇게도 가능

```
let s = 1;


for (let i = 1; i <6; i++) {

    s *= i

}
```

* 글자 뒤집기 재귀함수   
```
function reverse(txt) {

    txt += ''

    if (txt.length <= 1) {

        return txt;

    }

    return reverse(txt.slice(1)) + txt[0]

}
```
